# 如何參與貢獻 How to Contribute

## 文獻格式
採用 APA。

例如：
```
Chen, L., & Norman, J. 1965. **An introduction to the Foochow dialect**. San Francisco: San Francisco State College. 
```

中文作者名不縮寫。

例如：

```
李如龍, 梁玉璋, 鄒光椿, 陳澤平. 1994. **福州方言字典** 福州: 福建人民出版社
```
